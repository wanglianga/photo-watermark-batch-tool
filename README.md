# 📸 照片版权水印批处理工具

面向摄影师、修图师和图库运营的本地照片版权水印批量处理工具。所有处理均在本地完成，**照片不上传到任何远端服务器**，原图始终保留。

---

## 原始需求

> 请实现本地照片版权水印批处理工具，Electron 窗口面向摄影师、修图师和图库运营处理一批原片，照片不上传到任何远端。工具支持拖入文件夹、读取缩略图、查看或清理 EXIF、批量添加版权水印、按客户与拍摄主题重命名、压缩导出、写入处理日志和打开输出目录。用户能分别设置横图、竖图、方图的水印位置、透明度、Logo 尺寸和安全边距，先抽样预览再启动队列；队列显示当前文件、成功数、失败数、预计剩余时间和系统通知，原图必须保留。

## 扩展需求

> 扩展水印模板管理：用户可以把品牌名、Logo、透明度、边距、字体和位置保存成模板，下次拖入相同客户文件夹时自动推荐；导出前仍要抽样查看不同构图，防止水印压住人物脸部或商品主体。
> 扩展同名输出决策：目标目录已存在同名照片、只读文件或历史导出批次时，工具提供覆盖、跳过、自动编号、另选目录和按日期建子文件夹，任何选择都不能静默覆盖原文件。

## 🚀 启动方式

### 前置要求

- **Node.js** 版本 ≥ 18.0.0
- **npm** 或 **pnpm** 或 **yarn**（推荐 pnpm）
- **Windows / macOS / Linux** 桌面环境（Electron 应用）
- 如使用 Docker，需安装 **Docker Desktop** 或 **Docker Engine** + **Docker Compose**

### 启动步骤

#### 方式一：本地 Node.js 运行（推荐）

##### 1. 安装依赖

```bash
npm install
```

如果您在中国网络环境下，推荐使用国内镜像：

```bash
npm config set registry https://registry.npmmirror.com
npm install
```

> **注意**：`sharp` 包需要下载预编译二进制文件，如遇网络问题可设置环境变量：
> ```bash
> npm config set sharp_binary_host "https://npmmirror.com/mirrors/sharp"
> npm config set sharp_libvips_binary_host "https://npmmirror.com/mirrors/sharp-libvips"
> npm install
> ```

##### 2. 启动应用

```bash
npm start
```

启动后应用窗口会自动打开。

#### 方式二：Docker 运行（需桌面环境支持）

> **注意**：Electron 是 GUI 桌面应用，Docker 运行需要宿主机提供显示服务。
> - **Linux**：通过 X11 转发（见下方命令）
> - **Windows/macOS**：Docker 容器内无法直接渲染 GUI，建议使用方式一

##### 1. 复制环境变量示例文件

```bash
cp .env.example .env
```

##### 2. Linux X11 环境构建并启动

```bash
# 允许 Docker 访问 X 服务器
xhost +local:docker

# 构建并启动
docker compose up --build
```

后台运行：

```bash
docker compose up --build -d
```

停止并清理：

```bash
docker compose down
xhost -local:docker
```

Windows / macOS 用户请使用方式一（本地 Node.js 运行）。

## 🖱️ 使用指南

### 1. 导入图片
- 将文件夹或图片拖入左上区域的虚线框
- 或点击「选择文件夹」/「选择文件」按钮
- 支持格式：JPG、JPEG、PNG、TIFF、TIF、WebP、BMP

### 2. 查看信息
- 点击列表中任意文件，右侧会显示：
  - 预览图与尺寸信息
  - EXIF 拍摄参数（可展开查看全部）

### 3. 设置水印
**水印模板管理**：
- 点击下拉菜单选择已有模板，一键应用所有设置
- 输入模板名称后点击「保存」，将当前所有设置存为模板
- 选中模板后点击「删除」可移除模板
- 💡 拖入客户文件夹时，会自动推荐匹配的水印模板

**Logo 设置**：
- 选择 Logo 图片（推荐 PNG 透明背景）
- 填写文字水印（如：© 2024 摄影师名）
- 自定义颜色、描边、字体

**重命名设置**：
- 勾选「启用批量重命名」
- 填写客户名称（如：张三婚礼）、拍摄主题（如：外景主纱）
- 可选是否保留原文件名、添加序号、序号位数

**导出设置**：
- 选择输出目录（⚠️ 必须与原图不同目录）
- 输出格式：JPG / WebP / PNG / 保持原格式
- 压缩质量：1-100（越高质量越好体积越大）
- 可选「清理 EXIF」保护隐私
- 可选「调整尺寸」按长边/宽度/高度缩放

**方向参数设置**：
- 切换「横图 / 竖图 / 方图」三个标签页独立配置：
  - **Logo 位置**：9 个点位
  - **安全边距**：相对图片宽度/高度的百分比
  - **Logo 尺寸**：相对图片长边的百分比
  - **Logo 透明度**：1-100%
  - **文字位置 / 边距 / 尺寸 / 透明度**
- 点击「将当前设置复制到所有方向」一键同步

### 4. 抽样预览

**效果预览**：
- 点击「🔍 效果预览」按钮
- 弹窗左右对比原图与水印效果
- 满意后关闭弹窗进入下一步

**多构图抽样预览**：
- 点击「🖼️ 多构图抽样预览」按钮
- 一次性展示 6 种水印位置（左上、右上、左下、右下、上中、下中）
- 点击任意构图可直接应用该位置设置
- 使用「上一张 / 下一张」切换不同抽样图片
- 帮助检查水印是否遮挡脸部或商品主体
- 💡 建议在导出前使用此功能确认最佳水印位置

### 5. 开始批处理

**同名文件冲突处理**：
- 点击「▶️ 开始批处理」后，系统自动检测输出目录冲突
- 检测到同名文件时，弹出冲突处理对话框，提供 5 种策略：
  - **🔄 覆盖**：直接覆盖已存在的文件（只读文件会跳过）
  - **⏭️ 跳过**：不处理已存在的文件，保留原文件
  - **🔢 自动编号**：冲突文件自动添加序号后缀（如 _01、_02）
  - **📅 按日期建子文件夹**：在输出目录下创建当天日期子文件夹存放
  - **📂 另选目录**：重新选择一个输出目录
- ⚠️ **任何策略都不会静默覆盖原文件**，所有操作需用户确认

**进度查看**：
- 右侧「处理进度」面板实时显示：
  - 当前处理文件名
  - 进度百分比与进度条
  - ✓ 成功数 / ⏭ 跳过数 / ✗ 失败数
  - 预计剩余时间
- 中途可点击「⏹️ 取消处理」

### 6. 完成后
- 自动弹出系统通知
- 显示结果统计与文件列表
- 「📋 查看处理日志」打开日志文件
- 顶部「📁 打开输出目录」快速访问结果

## ⚠️ 注意事项

1. **原图保护**：处理结果写入输出目录，原图目录和文件**永不修改**
2. **输出目录**：建议选择与原图不同的目录，避免同名覆盖
3. **大数量处理**：超过 200 张会提示确认，处理中请勿关闭窗口
4. **Sharp 安装**：如需离线环境，可提前准备 sharp 的预编译二进制
5. **Docker 限制**：Electron GUI 无法在纯 headless 服务器环境运行
6. **内存占用**：处理超大图（>50MB）时建议单批次不超过 500 张

## 📝 日志说明

每次批处理会在**输出目录**生成日志文件：
```
处理日志_2024-06-15T14-30-00.txt
```

日志内容包含：
- 处理开始 / 结束时间
- 客户名、主题、输出目录
- 每张图的处理结果（成功 / 失败原因 / 输出体积 / 压缩比）
- 最终统计（成功数 / 失败数 / 总用时）
